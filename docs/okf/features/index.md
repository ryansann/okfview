# Features

* [Document view](document-view.md) - Rendered Markdown with a first-class frontmatter header, backlinks, and outgoing links.
* [Graph view](graph-view.md) - The concept link-graph, colored by type.
* [Search](search.md) - Full-text command palette across all open bundles.
* [Live sync](live-sync.md) - Bundles stay current as files change, without losing UI state.
* [Diagnostics](diagnostics.md) - OKF v0.1 conformance issues, surfaced not enforced.
* [Recents](recents.md) - Previously-imported bundles are remembered for one-click reopen.
* [Settings & the MCP dashboard](settings-and-mcp-dashboard.md) - In-app configuration and a first-class MCP control surface.
