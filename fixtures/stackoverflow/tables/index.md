# BigQuery Table

* [Badges](badges.md) - This table contains information about badges awarded to users on the Stack Overflow platform, including details about the badge name, class, and whether it is tag-based.
* [Comments](comments.md) - This table contains comments made by users on posts within the Stack Overflow platform, including details about the comment text, score, and author, along with licensing information.
* [Orphaned Tag Wiki Posts](posts_orphaned_tag_wiki.md) - This table contains Tag Wiki posts that are no longer associated with an active tag on Stack Overflow.
* [Post History](post_history.md) - Tracks the revision history of posts on Stack Overflow, including details about event types, associated users, and content changes.
* [Post Links](post_links.md) - Contains information about links between posts on Stack Overflow, including the type of link and the posts involved.
* [Posts Answers](posts_answers.md) - Contains information about answers to questions posted on Stack Overflow, including detailed schema fields, their descriptions, and licensing information.
* [Posts Moderator Nomination](posts_moderator_nomination.md) - Contains posts where users nominate themselves or others to become moderators on the Stack Overflow platform.
* [Posts Privilege Wiki](posts_privilege_wiki.md) - This table contains posts describing various privileges on Stack Overflow, such as editing questions and retagging.
* [Posts Tag Wiki](posts_tag_wiki.md) - Contains detailed wiki descriptions for tags on Stack Overflow.
* [Posts Tag Wiki Excerpt](posts_tag_wiki_excerpt.md) - Contains excerpts from tag wikis on Stack Overflow.
* [Posts Wiki Placeholder](posts_wiki_placeholder.md) - Placeholder table for Stack Overflow wiki posts, containing community-maintained help and information.
* [Stack Overflow Posts (Legacy)](stackoverflow_posts.md) - A legacy table containing all posts from Stack Overflow. This table is deprecated; use `posts_answers`, `posts_questions`, or other `posts_*` tables instead.
* [Stack Overflow Questions](posts_questions.md) - This table contains information about all question posts on Stack Overflow, including detailed schema fields, their descriptions, and licensing information.
* [Tags](tags.md) - A table containing information about tags used on Stack Overflow, including tag names, counts, and references to their excerpt and wiki posts, along with moderation settings.
* [Users](users.md) - This table contains information about users registered on Stack Overflow, including detailed profile information, activity metrics, and network-wide identifiers.
* [Votes](votes.md) - This table contains information about votes cast on posts within the Stack Overflow platform, including the vote type, associated post and user, and bounty details.
